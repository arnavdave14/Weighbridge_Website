# Weighbridge Automation Website

This is a premium, high-performance landing page for a Weighbridge Software Company.

## 🚀 Features
- **3D Hero Background**: Interactive particle system built with Three.js.
- **Modern UI**: Glassmorphism, smooth gradients, and premium typography (Outfit & Inter).
- **Advanced Animations**: Powered by GSAP (ScrollTrigger, ScrollTo).
- **Responsive**: Mobile-first design with a custom animated mobile menu.
- **Custom Cursor**: Interactive cursor that reacts to UI elements.

## 📁 Structure
- `index.html`: Main structure.
- `css/style.css`: Design system and styling.
- `js/main.js`: Animation and interaction logic.
- `assets/images/`: Images and assets.

## 🛠️ Usage
Open `index.html` in any browser to view the site.
